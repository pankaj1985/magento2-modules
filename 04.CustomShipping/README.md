# Pankaj_CustomShipping

## Purpose
This module adds a **custom shipping method** to Magento 2 checkout, allowing store owners to define a standard delivery option with configurable price and title.

## Goal
- Create a fully functional shipping carrier in Magento 2
- Provide admin configuration for enabling/disabling, setting method title and price
- Integrate seamlessly with Magento checkout
- Follow Magento 2 best practices for shipping modules

## Use Case
- Offer a standard/custom delivery option to customers
- Configure shipping cost directly from admin panel
- Extendable for future shipping logic or promotions
- Demonstrate Magento 2 custom shipping module development in interviews
