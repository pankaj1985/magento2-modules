# Pankaj_RestApiExample

## Purpose
This module is created to demonstrate how to build a custom REST API in Magento 2 using Web API and Service Contracts.

## Goal
- Expose a custom GET REST API endpoint
- Follow Magento 2 best practices (Interface + Repository pattern)
- Allow public (anonymous) access to the API
- Make the API easily extendable using Dependency Injection

## Use Case
- Fetch custom data using REST API
- Integrate Magento with external systems or mobile applications
- Use in headless Magento or third-party integrations
- Demonstrate REST API implementation in Magento 2 interviews
