\# Magento 2 Custom Modules Collection



This repository contains multiple custom Magento 2 modules developed to demonstrate real-world implementations and interview-ready examples.



\## Included Modules

\- Custom Shipping Method

\- Custom REST API Module

\- GraphQL (Public) Module

\- GraphQL with Authentication Module

\- Custom CLI Module

\- Custom Logger / Observer Module

\- Utility / Demo Module



\## Purpose

\- Showcase Magento 2 module development skills

\- Provide reusable and well-structured examples

\- Cover key Magento concepts like Shipping, REST API, GraphQL, DI, and Configuration



\## Usage

Each module is independent and contains its own README file with module-specific details.
